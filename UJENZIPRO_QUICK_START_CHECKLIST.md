## **🚨 TROUBLESHOOTING**

### **If ElevenLabs doesn't work:**
- Try Murf.ai as alternative
- Use free tier of Azure Cognitive Services
- Record your own voiceover with Audacity

### **If screen recording is choppy:**
- Close unnecessary applications
- Record in smaller segments
- Use OBS Studio instead of Loom
- Lower recording quality to 720p if needed

### **If app pages don't load:**
- Check if npm run dev is still running
- Clear browser cache and restart
- Try different browser (Chrome/Edge)
- Check console for JavaScript errors
