## ⚡ Quick Deploy (Copy & Paste These)

### Command 1: Login to Vercel
```bash
npx vercel login
```
- Browser will open
- Login/confirm
- Return to terminal

### Command 2: Link Project
```bash
npx vercel link
```
- Answer: Y (Yes) to prompts
- Select: UjenziPro from list

### Command 3: Deploy to Production
```bash
npx vercel --prod
```
- Uploads code
- Builds on Vercel
- Deploys to production
- **Done!** 🎉
