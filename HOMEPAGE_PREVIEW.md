## 🚀 HOW TO VIEW IT

### **Option 1: View on Netlify (Recommended)**
1. Go to https://app.netlify.com
2. Find your "UjenziPro2" site
3. Click on the site URL
4. See your live homepage!

### **Option 2: Install Node.js to View Locally**
If you want to view it locally in the future:

1. **Download Node.js:**
   - Go to: https://nodejs.org
   - Download the LTS version (v20.x)
   - Install it

2. **Restart PowerShell**

3. **Run the dev server:**
   ```bash
   npm run dev
   ```

4. **Open:** http://localhost:5173

- Fallback SVG is showing until then

**To Add Your Drone Image:**
1. Save the construction drone image as: 
   `public/construction-site-drones.jpg`
2. Commit and push:
   ```bash
   git add public/construction-site-drones.jpg
   git commit -m "Add drone background image"
   git push origin main
   ```
3. Netlify will auto-deploy with the new image!
