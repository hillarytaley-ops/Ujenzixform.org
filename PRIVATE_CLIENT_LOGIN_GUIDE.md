# 🏠 Private Client Login Guide - Step by Step

## 🎯 **How to Log In as a Private Client**

### **Option 1: You Already Have an Account**

1. **Go to the login page:**
   ```
   Visit: your-site.com/auth
   Or click "Sign In" in the navigation bar
   ```

2. **Click the "Sign In" tab**

3. **Enter your credentials:**
   ```
   Email: your-email@example.com
   Password: your-password
   ```

4. **Click "Sign In" button**

5. **✅ You're logged in!**
   - Redirected to homepage or dashboard
   - Can now purchase materials

---

### **Option 2: Create New Private Client Account**

#### **Step 1: Sign Up**

1. **Go to:** `your-site.com/auth`

2. **Click "Sign Up" tab**

3. **Enter:**
   ```
   Email: your-email@example.com
   Password: (min 8 characters)
   ```

4. **Click "Sign Up"**

5. **✅ Account created instantly!**
   - No email confirmation needed
   - Auto-logged in

#### **Step 2: Complete Private Client Profile**

After signup, you'll be directed to complete your profile:

1. **Go to:** `/private-client-registration`
   
   Or manually visit: `your-site.com/private-client-registration`

2. **Fill the form:**

   **Personal Information:**
   ```
   Full Name: John Kamau
   Phone: +254 712 345 678
   Location: Nairobi, Kenya
   ```

   **Project Details:**
   ```
   Property Type: Single Family Home
   Project Type: [Select one or more]
     ☑ New House Construction
     ☑ Home Renovation
     ☐ Kitchen Renovation
     etc.
   
   Budget Range: KSh 1,000,000 - KSh 2,500,000
   Timeline: 3-6 months
   
   Project Description:
   "Building a 3-bedroom house in Karen..."
   ```

3. **Click "Complete Registration"**

4. **✅ Profile complete!**
   - Redirected to Suppliers page
   - Can start shopping immediately

## ⚠️ **Troubleshooting:**

### **Issue: White page after signup**
**Solution:** 
- Disable email confirmation in Supabase
- See: `DISABLE_EMAIL_CONFIRMATION_SUPABASE.md`

### **Issue: Can't find Direct Purchase**
**Steps:**
1. Make sure you're logged in
2. Go to Builders page
3. Click "Builder Dashboard" button
4. Look for "Direct Purchase" tab
5. Click it
6. Click "Start Shopping for Materials"

### **Issue: Not seeing Materials**
**Solution:**
- Go directly to `/suppliers` page
- Browse materials there
- All purchasing happens on Suppliers page
