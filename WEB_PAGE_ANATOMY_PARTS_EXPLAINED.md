# 🎨 Web Page Anatomy - Parts of a Page Explained

**Understanding the various parts/sections of a web page**

---

## 📐 STANDARD WEB PAGE STRUCTURE

```
┌──────────────────────────────────────────────────────────┐
│  1. HEADER / NAVIGATION                                  │
│  ──────────────────────────                              │
│  Logo | Menu | Search | Login                            │

┌──────────────────────────────────────────────────────────┐
│  2. HERO SECTION                                         │
│  ────────────────                                        │
│  Main Heading                                            │
│  Subheading / Description                                │
│  Call-to-Action Buttons                                  │
│  Background Image                                        │
└──────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────┐
│  3. MAIN CONTENT AREA                                    │
│  ─────────────────────                                   │
```

## 🔍 DETAILED EXPLANATION OF EACH PART

### **1. HEADER / NAVIGATION BAR**

**Location:** Top of every page (usually sticky/fixed)

**Contains:**
```
┌──────────────────────────────────────────────────────────┐
│  HEADER / NAVBAR                                         │
├──────────────────────────────────────────────────────────┤
│  ┌────┐                                                  │
│  │LOGO│  Home  Builders  Suppliers  Delivery  [Sign In] │
│  └────┘                                                  │
└──────────────────────────────────────────────────────────┘

Parts of Header:
├─ Logo (brand identity, links to homepage)
├─ Primary Navigation Menu
│   ├─ Home
│   ├─ Products/Services
│   ├─ About
│   └─ Contact
├─ Secondary Navigation
│   ├─ Search bar
│   ├─ Shopping cart icon
│   ├─ User account
│   └─ Login/Signup buttons
└─ Mobile Menu (hamburger icon ☰)
```

**Purpose:**
- Navigate between pages
- Access key features
- User account access
- Branding (logo)

---

### **2. HERO SECTION**

**Location:** Top of page, just below header

**Contains:**
```
┌──────────────────────────────────────────────────────────┐
│  HERO SECTION                                            │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  ┌────────────────────────────────────────────────────┐ │
│  │                                                    │ │
│  │        🏗️ KENYA'S #1 CONSTRUCTION PLATFORM         │ │
│  │                                                    │ │
│  │     Connect with Verified Suppliers & Builders    │ │
│  │                                                    │ │
│  │     [Get Started]  [Browse Materials]             │ │
│  │                                                    │ │
│  │     Background: Beautiful construction image      │ │
│  │                                                    │ │
│  └────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────┘

Parts of Hero:
├─ H1 Heading (main title, large text)
├─ Subheading / Tagline
├─ Brief description
├─ Call-to-Action buttons (CTAs)
├─ Background image/video
├─ Overlay gradient
└─ Badge/Trust indicators
```

**Purpose:**
- Grab attention
- Communicate value proposition
- Encourage action (CTA buttons)
- Set visual tone

---

### **3. BREADCRUMBS** (Optional)

**Location:** Below header, above main content

**Contains:**
```
Home > Suppliers > Bamburi Cement > Cement 50kg
```

**Purpose:**
- Show current location
- Easy navigation up hierarchy
- Improve UX

### **5. MAIN CONTENT AREA**

**Location:** Center/majority of page

**Contains:**
```
┌──────────────────────────────────────────────────────────┐
│  MAIN CONTENT                                            │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  Section Title / Heading                                 │
│  ────────────────────────                                │
│                                                          │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐                 │
│  │ Card 1  │  │ Card 2  │  │ Card 3  │                 │
│  │ [Image] │  │ [Image] │  │ [Image] │                 │
│  │ Title   │  │ Title   │  │ Title   │                 │
│  │ Desc    │  │ Desc    │  │ Desc    │                 │
│  │ [Button]│  │ [Button]│  │ [Button]│                 │
│  └─────────┘  └─────────┘  └─────────┘                 │
│                                                          │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐                 │
│  │ Card 4  │  │ Card 5  │  │ Card 6  │                 │
│  └─────────┘  └─────────┘  └─────────┘                 │
│                                                          │
│  [Load More] [Pagination: 1 2 3 >]                      │
│                                                          │
└──────────────────────────────────────────────────────────┘

Common Content Types:
├─ Card Grid (products, suppliers, builders)
├─ Article/Blog content
├─ Forms (contact, checkout)
├─ Tables (data, orders)
├─ Lists (features, items)
├─ Images/Gallery
├─ Videos
└─ Interactive elements
```

**Purpose:**
- Display primary information
- Show products/services
- Present data
- User interactions

### **6. CARDS** (Most common UI element)

**Location:** Inside main content area

**Anatomy of a Card:**
```
┌──────────────────────────────┐
│  CARD                        │
├──────────────────────────────┤
│  ┌──────────────────────┐   │  ← Card Header
│  │ Card Title           │   │
│  │ Subtitle/Description │   │
│  └──────────────────────┘   │
│  ──────────────────────────  │
│  ┌──────────────────────┐   │  ← Card Body
│  │ [Image or Icon]      │   │
│  │                      │   │
│  │ Main content here    │   │
│  │ - Bullet points      │   │
│  │ - Paragraphs         │   │
│  │ - Data fields        │   │
│  │                      │   │
│  └──────────────────────┘   │
│  ──────────────────────────  │
│  [Action] [Button]           │  ← Card Footer
└──────────────────────────────┘

Card Parts:
├─ Card Header (title, subtitle)
├─ Card Body (main content)
├─ Card Footer (actions, buttons)
├─ Border/Shadow (visual separation)
└─ Hover effects
```

**Common Form Parts:**

```
┌──────────────────────────────────────────┐

├──────────────────────────────────────────┤

│  Form Title                              │
│  ───────────                             │
│                                          │
│  Label:                                  │

│                                          │
│  Label:                                  │

│                                          │

│  [────────────────────────────────────] │  ← Textarea
│  [                                    ] │
│  [                                    ] │
│                                          │
│  ☐ Checkbox label                       │  ← Checkbox
│                                          │
│  ○ Radio option 1                       │  ← Radio Button
│  ○ Radio option 2                       │
│                                          │
│  [Cancel]  [Submit]                     │  ← Action Buttons
│                                          │
└──────────────────────────────────────────┘

Form Parts:
├─ Form Title
├─ Field Labels
├─ Input Fields (text, email, phone, etc.)
├─ Dropdowns/Selects
├─ Textareas (multi-line)
├─ Checkboxes
├─ Radio Buttons
├─ File Upload
├─ Date Pickers
├─ Validation Messages (errors, success)
├─ Helper Text

```

### **8. TABLES**

**Common Table Structure:**

```
┌──────────────────────────────────────────────────────────┐
│  TABLE                                                   │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  ┌────────┬────────────┬──────────┬──────────┬────────┐ │
│  │ Header │ Header     │ Header   │ Header   │ Action │ │  ← Table Header
│  ├────────┼────────────┼──────────┼──────────┼────────┤ │
│  │ Cell 1 │ Cell 2     │ Cell 3   │ Cell 4   │ [Edit] │ │  ← Table Row
│  │ Cell 1 │ Cell 2     │ Cell 3   │ Cell 4   │ [Edit] │ │
│  │ Cell 1 │ Cell 2     │ Cell 3   │ Cell 4   │ [Edit] │ │
│  └────────┴────────────┴──────────┴──────────┴────────┘ │
│                                                          │
│  [< Previous] [1] [2] [3] [Next >]                      │  ← Pagination
│                                                          │
└──────────────────────────────────────────────────────────┘

Table Parts:
├─ Table Header (column titles)
├─ Table Body (data rows)
├─ Table Rows
├─ Table Cells
├─ Sort indicators (▲ ▼)
├─ Action column (buttons)
├─ Pagination
└─ Empty state (when no data)
```

### **9. MODAL / DIALOG**

**Overlay Component:**

```
┌──────────────────────────────────────────────────────────┐
│  FULL PAGE (Dimmed Background)                           │
│  ──────────────────────────────                          │
│                                                          │
│         ┌────────────────────────────────┐              │
│         │  MODAL / DIALOG                │              │
│         ├────────────────────────────────┤              │
│         │  ┌──────────────────────────┐ │              │
│         │  │ Modal Header         [×] │ │  ← Header    │
│         │  └──────────────────────────┘ │              │
│         │  ──────────────────────────── │              │
│         │  ┌──────────────────────────┐ │  ← Body      │
│         │  │ Modal content here       │ │              │
│         │  │ - Text                   │ │              │
│         │  │ - Form fields            │ │              │
│         │  │ - Images                 │ │              │
│         │  └──────────────────────────┘ │              │
│         │  ──────────────────────────── │              │
│         │  [Cancel] [Confirm]           │  ← Footer    │
│         └────────────────────────────────┘              │
│                                                          │
└──────────────────────────────────────────────────────────┘

Modal Parts:
├─ Backdrop/Overlay (dimmed background)
├─ Modal Container
├─ Modal Header
│   ├─ Title
│   └─ Close button (×)
├─ Modal Body
│   └─ Content (text, forms, images)
├─ Modal Footer
│   ├─ Cancel button
│   └─ Confirm/Submit button
└─ Close triggers (×, backdrop click, Esc key)
```

## 🏗️ COMMON PAGE SECTIONS

### **1. ABOVE THE FOLD**

**Parts visible without scrolling:**
```
- Header/Navigation
- Hero Section
- First part of main content
```

### **2. BELOW THE FOLD**

**Parts requiring scroll:**
```
- Rest of main content
- Features sections
- Testimonials
- CTA sections
- Footer
```

---

### **3. STICKY ELEMENTS**

**Parts that stay visible on scroll:**
```
- Header (sticky navigation)
- Floating action buttons
- Chat widget
- Back to top button
- Cookie banner
```

---

### **4. OVERLAY ELEMENTS**

**Parts that appear on top:**
```
- Modals/Dialogs
- Dropdowns
- Tooltips
- Toast notifications
- Loading spinners
- Context menus
```

## 📋 STANDARD TERMS GLOSSARY

### **Page Structure Terms:**

| Term | Definition | Example |
|------|------------|---------|
| **Header** | Top section with navigation | Logo + Menu |
| **Hero** | Large banner section at top | Main title + CTA |
| **Sidebar** | Side panel with filters/links | Filter options |
| **Main Content** | Primary page content | Cards, articles, products |
| **Footer** | Bottom section with links | Company info, links |
| **Above the Fold** | Visible without scrolling | Hero + first content |
| **Below the Fold** | Requires scrolling to see | Lower content |
| **CTA** | Call-to-Action button/section | "Get Started" button |

---

### **UI Component Terms:**

| Term | Definition | Example |
|------|------------|---------|
| **Card** | Container for grouped content | Product card, profile card |
| **Modal** | Overlay dialog box | Popup form, confirmation |
| **Dropdown** | Expandable selection menu | Category selector |
| **Accordion** | Expandable/collapsible sections | FAQ sections |
| **Tabs** | Switch between content panels | Product details tabs |
| **Badge** | Small label/tag | "New", "Verified", "Sale" |
| **Tooltip** | Hover information popup | Icon explanations |
| **Toast** | Temporary notification | "Success!", "Error!" |
| **Breadcrumb** | Navigation trail | Home > Products > Item |
| **Pagination** | Page navigation controls | [< 1 2 3 >] |

---

### **Form Element Terms:**

| Term | Definition | Example |
|------|------------|---------|
| **Label** | Field description text | "Email Address:" |
| **Input** | Single-line text field | [____________] |
| **Textarea** | Multi-line text field | Large text box |
| **Checkbox** | Multiple selection option | ☐ Remember me |
| **Radio** | Single selection option | ○ Option 1 |
| **Select** | Dropdown list | [Choose one ▼] |
| **Button** | Clickable action | [Submit] |
| **Placeholder** | Example text in field | "Enter your email" |
| **Validation** | Error/success message | "Email is required" |
