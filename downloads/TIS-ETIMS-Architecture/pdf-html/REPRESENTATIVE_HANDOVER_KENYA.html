Purpose

You are helping submit a read-only CD or DVD to the Kenya Revenue Authority (KRA) as part of UjenziXform’s application to become a third-party Trader Invoicing System (TIS) integrator for eTIMS.

You do not need to be technical. Your main tasks are:

  1. Burn a read-only disc from files we send you
  2. Print and sign the cover letter
  3. Deliver the disc and letter to KRA
  4. Send proof of submission back to us

Step 1 — Download and unzip

  1. Download the zip file we send you.
  2. Right-click → Extract All (or use WinRAR / 7-Zip).
  3. Open the extracted TIS-OPTICAL-DISK folder.
  4. Confirm you see 01-SOFTWARE, 02-PDF-DOCUMENTS, and README.txt.

Quick check: Open 02-PDF-DOCUMENTS — there should be 8 PDF files (numbered 01 to 08).

Step 2 — Burn a read-only CD or DVD

Option A — CD/DVD shop (recommended if you have no burner)

  1. Copy the entire TIS-OPTICAL-DISK folder onto a USB stick.
  2. Go to any computer / IT / printing shop in Nairobi (or your town) that burns discs.
  3. Give them the USB stick and this instruction slip:
Please burn a DATA CD or DVD.

Copy everything INSIDE the TIS-OPTICAL-DISK folder to the disc root:
  - README.txt
  - SOFTWARE_MANIFEST.txt
  - 01-SOFTWARE (folder)
  - 02-PDF-DOCUMENTS (folder)
  - 03-SOURCE-DOCUMENTS (folder, if present)

Requirements:
  - Finalized / read-only (not rewritable)
  - No multi-session
  - Make 2 copies

Label each disc:
  UjenziXform TIS v1.0.0
  UJX-TIS-KRA-ARCH-2026-001
  KRA eTIMS TIS Certification — Read Only
  Date: [today's date]
  1. Pay for 2 copies — one for KRA, one for our records.
  2. Before leaving the shop, open the disc on their PC and confirm files open (especially the PDFs).

Option B — Burn yourself (Windows)

  1. Insert blank CD-R or DVD-R.
  2. Open the TIS-OPTICAL-DISK folder.
  3. Select all items inside (not the parent folder name alone).
  4. Right-click → Send to → DVD RW Drive.
  5. Choose With a CD/DVD player if Windows asks.
  6. Click Burn to disc.
  7. Eject and re-insert — confirm you cannot delete files on the disc (read-only).
  8. Repeat for a second backup copy.

Step 3 — Print and sign the cover letter

  1. Open 02-PDF-DOCUMENTS\01_KRA_TIS_COVER_LETTER.pdf.
  2. Read it first — all details should already be filled in by our team in Australia.
  3. If any field still shows brackets like [Complete before submission], contact us before printing.
  4. Print on company letterhead if available.
  5. The authorised signatory signs and dates at the bottom.

Keep one printed copy for our records if possible.
