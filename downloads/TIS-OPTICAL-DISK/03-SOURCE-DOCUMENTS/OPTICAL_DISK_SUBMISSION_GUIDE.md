## 4. Steps to complete before submitting the disk

1. **Review** all seven PDFs in `02-PDF-DOCUMENTS/` (regenerate with `scripts/generate-tis-cert-pdfs.ps1` if sources change).
2. **Re-run** `powershell -File scripts/build-tis-optical-disk.ps1` if TIS source changed since last build.
3. **Review** `SOFTWARE_MANIFEST.txt` — confirm file count and no secrets.
4. **Copy** this architecture pack into `TIS-OPTICAL-DISK/03-SOURCE-DOCUMENTS/` (optional Markdown originals).
5. **Fill in** cover letter bracketed fields and test plan sign-off table.
6. **Burn** to CD/DVD as **read-only** (or submit read-only ISO if KRA accepts).
7. **Label disc:** UjenziXform TIS v1.0.0 — KRA Certification Submission — UJX-TIS-KRA-ARCH-2026-001.
8. **Submit** to KRA with cover letter **before** certification testing.
