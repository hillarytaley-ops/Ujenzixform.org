### **Step 2: Audio Recording**

#### **Narration Recording Setup**
```yaml
Recording Environment:
  - Quiet room with minimal echo
  - Consistent background noise level
  - Good microphone positioning (6-8 inches)
  - Pop filter to reduce plosives
  - Comfortable speaking position

Audio Settings:
  - Sample Rate: 48kHz
  - Bit Depth: 24-bit
  - Format: WAV (uncompressed)
  - Input Level: -12dB to -6dB peak
  - Monitor with headphones
```

#### **Narration Script Delivery**
```yaml
Speaking Guidelines:
  - Clear, professional tone
  - Moderate pace (150-160 words per minute)
  - Kenyan accent preferred
  - Emphasize key phrases
  - Natural pauses for visual elements
  - Consistent energy throughout

Recording Process:
  - Record in segments matching video scenes
  - Multiple takes per segment
  - Room tone recording (30 seconds silence)
  - Backup recordings on separate device
  - Immediate playback review
```

### **Step 3: Music & Sound Effects**

#### **Background Music Integration**
```yaml
Music Selection Criteria:
  - Uplifting, modern with African influences
  - 4+ minutes duration (longer than video)
  - Royalty-free or properly licensed
  - Instrumental (no competing vocals)
  - Dynamic range suitable for mixing

Audio Processing:
  - Normalize to -23 LUFS (broadcast standard)
  - EQ to complement narration frequencies
  - Compression for consistent levels
  - Fade in/out at appropriate points
```

#### **Sound Effects Library**
```yaml
Required Sound Effects:
  - UI interaction sounds (clicks, hovers)
  - Transition whooshes
  - Success notification chimes
  - Counter animation beeps
  - Achievement unlock sounds
  - Ambient construction site audio (subtle)

Processing Guidelines:
  - Short duration (0.1-0.8 seconds)
  - Consistent volume levels
  - EQ to fit in mix
  - Stereo positioning for depth
```

## 📈 Analytics Implementation

### **Video Analytics Setup**
```javascript
// Google Analytics 4 video tracking
function trackVideoEvents() {
  const video = document.querySelector('#ujenzipro-demo-video');
  
  // Video start
  video.addEventListener('play', () => {
    gtag('event', 'video_start', {
      video_title: 'UjenziPro Home Page Demo',
      video_duration: 225
    });
  });
  
  // Video completion
  video.addEventListener('ended', () => {
    gtag('event', 'video_complete', {
      video_title: 'UjenziPro Home Page Demo',
      video_duration: 225
    });
  });
  
  // Progress milestones
  video.addEventListener('timeupdate', () => {
    const progress = (video.currentTime / video.duration) * 100;
    
    if (progress >= 25 && !video.milestone25) {
      gtag('event', 'video_progress', {
        video_title: 'UjenziPro Home Page Demo',
        progress: 25
      });
      video.milestone25 = true;
    }
    
    // Similar for 50%, 75%
  });
}
```
