## 🎨 Visual Style Guide

### **Color Palette**:
- **Primary**: UjenziPro brand blue (#2563eb)
- **Kenyan Green**: #006633 (flag green)
- **Acacia Gold**: #FFD700 (warm, premium)
- **Construction Orange**: #FF6B35 (energy, action)
- **Safety Yellow**: #FFEB3B (attention, caution)
- **Kenyan Black**: #000000 (flag black)

### **Typography**:
- **Headlines**: Bold, modern sans-serif
- **Body Text**: Clean, readable sans-serif
- **Overlays**: High contrast, easy to read
- **Sizes**: Responsive to screen recording resolution

### **Animation Principles**:
- **Timing**: Smooth, not too fast or slow
- **Easing**: Natural, organic movement
- **Consistency**: Similar animations for similar elements
- **Purpose**: Every animation serves a function

### **Branding Elements**:
- **Logo**: Consistent placement and sizing
- **Colors**: Brand-consistent throughout
- **Voice**: Professional yet approachable
- **Imagery**: Authentic Kenyan construction context

## 📱 Technical Production Notes

### **Screen Recording**:
- **Resolution**: 1920x1080 minimum
- **Frame Rate**: 30fps (60fps for smooth UI interactions)
- **Browser**: Chrome for consistent rendering
- **Zoom Level**: 100% for crisp text
- **Cursor**: Highlighted for visibility

### **Audio Production**:
- **Narration**: Professional voice talent
- **Music**: Licensed, culturally appropriate
- **Sound Effects**: Subtle, purposeful
- **Mixing**: Balanced levels, clear dialogue

### **Post-Production**:
- **Editing Software**: Professional grade (Premiere Pro, Final Cut)
- **Color Grading**: Consistent, brand-appropriate
- **Motion Graphics**: Professional overlays and animations
- **Compression**: Optimized for web delivery

### **Quality Assurance**:
- **Preview**: Multiple device/screen size testing
- **Audio**: Clear on various playback systems
- **Loading**: Fast loading across connection speeds
- **Accessibility**: Captions and audio descriptions
